# 📱 GateVue App (Visitor Management System)

## 🚀 Overview

This is a React Native mobile application inspired by a real-world visitor management system (GateVue). The app replicates core functionalities such as authentication, role-based navigation, and visitor tracking.

## 🎯 Objective

To build a real-world mobile application with multiple user roles and dynamic navigation using React Native.

## ✨ Features

* 🔐 User Login (API integrated)
* 👥 Role-Based Access:

  * Admin
  * Security
  * Super Admin
* 🧭 Dynamic Navigation based on roles
* 📋 Visitor Management
* 📊 Dashboard & Reports

## 🧠 Implementation Highlights

* Role-based navigation using Context API
* Persistent login using AsyncStorage
* API handling using Axios
* Clean and modular folder structure

## 🛠 Tech Stack

* React Native
* JavaScript
* Axios
* AsyncStorage
* React Navigation

## 📂 Project Structure

src/
 ├── assets/        # Images and icons
 ├── components/    # Reusable UI components
 ├── screens/       # App screens
 ├── navigation/    # Navigation setup
 ├── services/      # API calls (Axios)
 ├── context/       # Global state management
 ├── constants/     # App constants
 └── hooks/         # Custom hooks (future use)
